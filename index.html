Meet Up That Way

Where introvert's gatherings come to life with a click! Our platform is a masterful clone of MeetUp, engineered with the latest web technologies to give users full CRUD capabilities for their Groups and Events. Not just that, users can also add pizaz to their gatherings with Create and Read functionalities for group and event images.

Frontend creativity meets backend brawn as we harness the reactive prowess of React and Redux. This dynamic duo makes a better combo than Reese's could ever do, this combo brings a state-of-the-art user experience that's lightning-fast, reducing reliance on slow snail database queries with each navigation. React Router keeps the transitions smooth, while CSS and Font Awesome sprinkles tad bit of style and icons to hold my users’ eyes.

The backend is a fortress built on the strong foundations of Node.js and Express, with Express-Validator ensuring that the data integrity of my users are kept. Sequelize is my chef who takes my query and data-types as ingredients and recipe, and gives me an effective and fast dish. My dish being my results. JWT fortifies my user's sessions.

Join MeetUpThatWay, where every click is a step into the future of introvert's socializing—fast, secure, and stunningly intuitive.

  • Backend — Node.js, Express, Express-Validator, Sequelize, JWT, PostgreSQL
  • Frontend — React, React Router, Redux, CSS, Font Awesome

Space Odyssey (Group Project)

Imagination finally meets meticulous planning, and collaboration pushes us beyond the final frontier. This platform is the testimony for human ingenuity, offering a digital cosmos where collective brainstorming reaches new heights.

Users will find themselve flying through the vastness of project planning with four full CRUD functionalities, connecting grand visions onto Boards and Lists, and refining smaller details on Cards. The experience is further enriched with image uploads specific to each card, fostering a collaborative spirit that elevates interaction to its zenith.

Under the hood of this collaboration starship, Space Odyssey is powered by a constellation of cutting-edge technologies.

A RESTful API crafted with Python and Flask manages user authentication and content management.
SQLAlchemy models provides structure to the data universe, while Alembic migration scripts navigate the evolving database schema.
The user interface built on the stellar of React components, is brought to life with Redux ensuring that data is accurately beamed to the application's state and database, ensuring a quick and responsive user journey.
React Router facilitates seamless navigation across this digital galaxy, and AWS integration ensures that image uploads are as effortless as stargazing.
Docker encapsulates the essence of our application into portable containers, ensuring consistent deployment across any environment, from development machines to the far reaches of cloud infrastructure.

  • Backend — Flask, SQLAlchemy, Alembic, Flask-Migrate, Flask-CORS, Flask-Login, Flask-SQLAlchemy, Flask-WTF, Jinja2, Mako, Werkzeug, WTForms, Boto3, AWS S3,, BCrypt, PostgreSQL
  • Frontend — React, React Router, Redux, CSS, Font Awesome
  • Repo

EverScrolls

Charting a course through the sea of ideas, where inspiration meets organization. This platform is a beacon for navigators of knowledge, offering a digital haven where seafarers of thought can anchor their musings.

Mariners embark on a voyage of discovery with three full CRUD functionalities, steering their course with Notebooks (Mariner's Journal), charting the tides with Notes (Mariner's Pages), and logging their journey with Tasks (Captain’s Log). Each feature is a compass point guiding users through the waves of creativity and collaboration.

Below deck, EverScrolls is steered by a crew of robust technologies.

A RESTful API, crafted with the precision of a shipwright, manages user authentication and content management.
SQLAlchemy models serve as the sturdy hull, providing structure to the data ocean, while Alembic migration scripts are the navigational charts that guide the evolving database schema.
The user interface, designed with the elegance of a captain's quarters, is built upon the sturdy beams of React components, with Redux as the wind in the sails ensuring that data flows smoothly to the application's state and database for a swift and responsive user experience.
